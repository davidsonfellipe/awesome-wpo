# Contributing

Thanks for helping improve this list.

## How to contribute

1. Open a [pull request](https://github.com/davidsonfellipe/awesome-wpo/pulls) or an [issue](https://github.com/davidsonfellipe/awesome-wpo/issues) if you want to discuss something first.
2. Add new resources under the right section in `README.md`, using this pattern:

   `- [Name](https://example.com/) - Short description that ends with a period.`

3. Prefer one clear link and description per line. Avoid duplicate links unless the context is clearly different.

Before you submit, run `pnpm awesome-lint` (or `npm run awesome-lint`) so `README.md` stays consistent with [awesome-lint](https://github.com/sindresorhus/awesome-lint) rules.
